## 关系模型与文档模型

### NoSQL

并非特定技术，(Not only SQL)

+ 比关系型数据库更好的可扩展性，包括非常大的数据集或非常高的写入吞吐量
+ 开源
+ 关系模型不能很好支持特殊的查询
+ 更具多动态性与表现力的数据模型

### 对象关系不匹配

如果数据存储在关系表中，那么需要一个笨拙的转换层，处于应用程序代码中的对 象和表，行，列的数据库模型之间

> "阻抗不匹配":每个电路的输入和输出都有一定的阻抗（交流电阻）
>
> 将一个电路的输出连接到另一个电路的输入时，如果两个电路的输出和输入阻抗匹配， 则连接上的功率传输将被最大化。阻抗不匹配会导致信号反射及其他问题。
